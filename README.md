## GPS 座標輸入格式

「GPS 座標」欄位支援以下各種格式：

### Decimal Degrees（十進位度數）

| 格式 | 範例 |
|------|------|
| 逗號分隔（有無空格皆可） | `25.006003,121.454933` |
| 逗號分隔 + 括號 | `(25.006003, 121.454933)` |
| 分號分隔 | `25.006003; 121.454933` |
| 空白分隔 | `25.006003 121.454933` |
| 方位字母在前 | `N25.006003 E121.454933` |
| 方位字母在後 | `25.006003N 121.454933E` |
| 方位 + 空白 + 數字 | `N 25.006003, E 121.454933` |
| 帶度符號 | `25.006003°N 121.454933°E` |

### Degrees Decimal Minutes（度分）

| 格式 | 範例 |
|------|------|
| 方位在前 | `N25° 0.36018' E121° 27.2960'` |
| 方位在後 | `25° 0.36018'N 121° 27.2960'E` |

### Degrees Minutes Seconds（度分秒）

| 格式 | 範例 |
|------|------|
| 方位在前，緊湊 | `N25°0'21.61" E121°27'17.76"` |
| 方位在後 | `25°0'21.61"N 121°27'17.76"E` |
| 方位在前，有空格 | `N25° 0' 21.61" E121° 27' 17.76"` |
| 方位在後，有逗號 | `25° 0' 21.61" N, 121° 27' 17.76" E` |

### 其他

| 格式 | 範例 |
|------|------|
| geo URI | `geo:25.006003,121.454933` |
| Google Maps URL 參數 | `@25.006003,121.454933` |

> Unicode 符號（`˚` `º` `′` `ʹ` `″` 等）會自動正規化，不影響辨識。
