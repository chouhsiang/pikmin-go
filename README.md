# 皮皮 iOS 座標模擬器

在瀏覽器上操控 iOS 裝置的 GPS 模擬位置，搭配本機後端（pymobiledevice3）即時更新裝置座標。

---

## 介面結構

側邊欄分為兩個主功能，透過左側導覽列切換：

- **打菇** — 設定裝置 GPS 座標
- **種花** — 定時自動移動

點擊側邊欄底部的收合按鈕可將側邊欄縮為導覽列，讓地圖視野最大化。

---

## 功能

### 設備連線

- 透過 **tunneld** 偵測已連線的 iOS 裝置，支援有線與無線連線
- 下拉選單顯示所有可用設備，點擊重整圖示可重新掃描

### 打菇 — 座標設定

三種方式可切換（分頁標籤）：

1. **好菇點** — 常用據點快選（開發中）
2. **純點** — 從預設座標清單（`locations.json`）以 pill 按鈕形式快選
3. **手動輸入** — 貼上座標後按 Enter 或點擊移動按鈕

**點選地圖** 也可直接將任意位置設為模擬座標。

**「回到現實位置」** — 讀取裝置的真實 GPS 位置，將地圖與模擬座標一併同步至現實位置。

### 支援的 GPS 輸入格式

#### Decimal Degrees（十進位度數）

| 範例 |
|------|
| `25.006003,121.454933` |
| `(25.006003, 121.454933)` |
| `25.006003; 121.454933` |
| `25.006003 121.454933` |
| `N25.006003 E121.454933` |
| `25.006003N 121.454933E` |
| `25.006003°N 121.454933°E` |

#### Degrees Decimal Minutes（度分）

| 範例 |
|------|
| `N25° 0.36018' E121° 27.2960'` |
| `25° 0.36018'N 121° 27.2960'E` |

#### Degrees Minutes Seconds（度分秒）

| 範例 |
|------|
| `N25°0'21.61" E121°27'17.76"` |
| `25°0'21.61"N 121°27'17.76"E` |
| `N25° 0' 21.61" E121° 27' 17.76"` |
| `25° 0' 21.61" N, 121° 27' 17.76" E` |

#### 其他

| 範例 |
|------|
| `geo:25.006003,121.454933` |
| `@25.006003,121.454933` |

> Unicode 度分秒符號（`˚` `º` `′` `ʹ` `″` 等）會自動正規化，不影響辨識。

### 種花 — 定時移動

讓裝置依指定速度、方向、時長自動移動：

- **時速**（km/h）— 設定移動速度，hover 資訊圖示可查看換算後的 m/s
- **持續時間**（分鐘）— 設定移動總時長
- **方位** — 東 / 南 / 西 / 北四個方向
- 按下 ▶ 開始後，地圖上會即時標出每秒的移動軌跡點；按 ⏸ 隨時停止

前端依參數產生 GPX 路線，後端透過 pymobiledevice3 播放，裝置在系統層面移動。
